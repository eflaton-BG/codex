# Voice Reference

Use these examples to match Ezekiel Flaton's assignment style in `#rps-vrs`.

## Examples

`<@U01UZ84CKH6|malhar> , you're up for the Thursday VRR.  Can you annotate today's 4/21 run?`

`For the Thursday VRR:`

`<@U05DLSQMGP5|abinav.anantharaman> can you take the 3/14 ITF run?  That was on your integration branches.`

`<@U01KUDHJMLY|Danny Chua> can you take the 3/15 ITF run (from today)`

`Plan for the VRR today`

`<@U01UZ84CKH6|malhar> You're on the 4/21 run`

`<@U05DLSQMGP5|abinav.anantharaman> can you take today's mini-vr that <@U068XMN6ZPA|Parvathy Ajith> posted above?  It was 20 minutes so it should be a short annotation.`

## Style Summary

- Lead with the assignment, not background
- Use one sentence when possible
- Mention the run date directly
- Mention the site if relevant
- Use `can you take`, `you're on`, or `you're up`
- Avoid long explanations
- Avoid sounding managerial or formal
- Keep punctuation simple

## Recommended Default Draft

Use this unless the channel context suggests a better variation:

`<@USER> you're up for the next VRR. Can you annotate the <RUN DATE> <SITE> run?`
